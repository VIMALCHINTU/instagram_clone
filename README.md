# Instagram Clone

A full-stack Instagram clone that allows users to create accounts, upload posts, like posts, and comment on posts.  
The project replicates core social media features while demonstrating full-stack development skills.

## Features

- User authentication (Signup / Login)
- Create and upload posts
- Like and comment on posts
- View user profiles
- Responsive UI
- Secure API integration

## Tech Stack

Frontend:
- React.js
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Other Tools:
- Git
- GitHub
